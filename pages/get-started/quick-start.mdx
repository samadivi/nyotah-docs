# Getting Started Guide with Nyotah Cloud

Welcome to the Nyotah Cloud getting started guide! This guide will help you get started with our cloud computing platform and deploy your first virtual machine in a few simple steps.

## Step 1: Create an Account

If you don't already have a Nyotah Cloud account, you'll need to create one to get started. Follow these steps to create your account:

1. Go to our website [nyotahcloud.com](https://www.nyotahcloud.com).
2. Click on the "Sign Up" or "Create an Account" button.
3. Fill out the registration form with your personal information.
4. Verify your email address by following the confirmation link sent to your inbox.

Once your account is created, you can log in to Nyotah Cloud using your credentials.

## Step 2: Access the Dashboard

Once logged in to your account, you'll be directed to the Nyotah Cloud dashboard. This is where you can manage your virtual resources, deploy new virtual machines, and access other platform features.

## Step 3: Deploy a Virtual Machine (VM)

Now that you're set up, it's time to deploy your first virtual machine. Follow these steps to deploy a VM:

1. On the dashboard, click on the "New VM" or "Deploy VM" button.
2. Choose the type of virtual machine you want to create, for example, Ubuntu, CentOS, Windows, etc.
3. Configure the VM options, such as memory size, storage space, etc.
4. Click "Deploy" to start the VM creation process.

Once the deployment process is complete, your new virtual machine will be ready to use.

## Step 4: Connect to the Virtual Machine

Now that your VM is deployed, you can connect to it to start working. Here's how to connect to your VM:

1. On the dashboard, find your VM in the list of virtual machines.
2. Click on the VM name to view its details.
3. Find the VM's IP address and connection information (username, password).
4. Use an SSH client or remote desktop software to connect to the VM using this information.

Once connected, you can start configuring and using your VM as needed.
